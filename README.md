# LinkerAI: Autonomous Multi-Portal Job Application Agent

LinkerAI is a premium, modern, self-hosted AI agent designed to automate the job application lifecycle. It uses large language models (via AWS Bedrock/LangChain) and browser automation (via Playwright) to scan job boards, compute real-time ATS match scores, write personalized LinkedIn/outreach drafts, and autonomously submit applications.

---

## 🚀 Key Features

1. **Dynamic AI Career Analysis**
   - Extracts your top skills, estimated years of experience, and recommended job roles directly from your uploaded resume.
   - Powered by AWS Bedrock for high-accuracy parsing and intelligence.
   
2. **Interactive Job Targeting**
   - Click recommended roles generated by the AI to immediately add them to your search keywords.
   - Configure target location (e.g. Remote, Bengaluru, Chennai), date posted/recency, and experience requirements.

3. **Multi-Portal Job Automation**
   - Supports multiple major job portals (LinkedIn, Indeed, Naukri, Foundit, Internshala, Unstop, Instahyre).
   - Log-in required portals are fully supported.

4. **Autopilot & Smart Limits**
   - **Auto-Apply Mode**: Turn on auto-apply to submit applications autonomously.
   - **Limit Selector**: Choose to apply to 10, 20, 30, 40, or a maximum of 50 jobs per run to prevent portal rate limits.
   
5. **Live Activity Terminal**
   - Watch step-by-step agent logs in real time as the browser engine performs navigation, search, and validation.

6. **Unified Application Tracker**
   - Track successful submits, drafted messages, ATS match scores, recruiter contacts, and pending follow-ups in a beautiful database layout.

---

## 🛠️ Architecture & Tech Stack

- **Frontend**: React, Vite, Tailwind CSS / Vanilla CSS, Lucide Icons, Axios, React Router.
- **Backend**: FastAPI (Python), SQLAlchemy ORM, Uvicorn.
- **Database**: SQLite (isolated per-user mapping).
- **AI Core**: LangChain, AWS Bedrock (default: `minimax.minimax-m2.5` or OpenAI compatible provider).
- **Automation**: Playwright Browser Engine.

## ⚙️ Environment Configuration

Create a `.env` file in the root of the project:

```env
# Backend Keys
BEDROCK_API_KEY=your_aws_bedrock_api_key
BEDROCK_API_URL=https://api.bedrock.us-east-1.amazonaws.com  # or compatibility URL
BEDROCK_MODEL=minimax.minimax-m2.5

# Server Ports
PORT=8000
```

## 📦 Installation & Getting Started

### 1. Prerequisites
- Python 3.10+
- Node.js 18+

### 2. Backend Setup
1. Open a terminal and navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Create and activate a Python virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
3. Install dependencies:
   ```bash
   pip install -r ../requirements.txt
   ```
4. Perform database migrations and setup:
   ```bash
   python migrate_db.py
   ```
5. Run the server:
   ```bash
   uvicorn app.main:app --reload --port 8000
   ```

### 3. Frontend Setup
1. Open another terminal and navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install npm packages:
   ```bash
   npm install
   ```
3. Launch the Vite development server:
   ```bash
   npm run dev
   ```

Open `http://localhost:5173` in your browser to start your AI-powered job search!

```
[Resume PDF] -> [AWS Bedrock Engine] -> [Skills, Roles, Experience]
                                               |
                                     (Click to Target Roles)
                                               |
                                               v
[Job Search Criteria] -> [Playwright Engine] -> [Extract Job Details]
                                                        |
                                            [ATS Match Scoring Engine]
                                                        |
                                          +-------------+-------------+
                                          |                           |
                                    (ATS Score >= 85%)          (ATS Score < 85%)
                                          |                           |
                                          v                           v
                               [Autonomously Submit]          [Skip Application]
                               [Log Recruiter Contacts]       [Log Recommendations]
```
