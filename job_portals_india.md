# Tech & IT Job Portals in India

A comprehensive directory of job application portals and talent platforms in India specializing exclusively in Tech, Software Engineering, IT, and Startup roles.

---

## 1. Core Tech & Software Engineering Portals
These platforms are designed specifically for software developers, product managers, UI/UX designers, QA, and data professionals. They offer low noise and direct recruiter matching.

| Portal | Target Audience | Key Features | URL |
| :--- | :--- | :--- | :--- |
| **Instahyre** | Software Engineers, QA, Devops, Product | AI-based matchmaking; zero-spam policy; matches candidates with top tech companies and startups. | [instahyre.com](https://www.instahyre.com) |
| **CutShort** | Software Developers, Tech Leads, PMs | AI-driven search and matching; direct chat with founders/hiring managers; highly transparent. | [cutshort.io](https://cutshort.io) |
| **Hirist** | Software Engineers, Data Scientists, DevOps | A dedicated job board exclusively for IT and technology roles across India. | [hirist.tech](https://hirist.tech) |
| **Geektrust** | Software Engineers & Developers | Candidates submit code matching a challenge; companies request interviews based on code quality. | [geektrust.in](https://www.geektrust.in) |
| **TechGig** | Developers, Coders & Freshers | Combines job listings with coding contests, hackathons, and skill assessments. | [techgig.com](https://www.techgig.com) |

## 5. Major General Portals (Tech Sections Only)
Although general, these remain massive sources of tech and IT jobs if filtered properly.

*   **LinkedIn Jobs India ([linkedin.com](https://www.linkedin.com)):** Critical for professional networking, direct cold-outreach to engineering managers, and tracking tech job posts.
*   **Naukri.com (IT & Software Category) ([naukri.com](https://www.naukri.com)):** Remains the largest aggregator of IT services roles (TCS, Infosys, Cognizant, Accenture, etc.) in India.
*   **Indeed India (Tech Filters) ([indeed.co.in](https://www.indeed.co.in)):** Useful for aggregating tech vacancies from corporate sites and portals.
